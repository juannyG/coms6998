# Setup

```sh
$ git clone git@github.com:juannyG/coms6998.git
$ cd coms6998
$ yarn install
$ cp packages/foundry/.env.example packages/foundry/.env
$ yarn compile
$ yarn chain

# Open a new tab
$ yarn deploy

# Open a new tab
$ yarn start
```

To interact with spotlight, go to http://localhost:3000/spotlight
