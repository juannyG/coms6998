# Setup

```sh
$ python3 bin/spotlight-cli.py local build --run

# Open a new tab - deploy the contract to your local test net
$ python3 bin/spotlight-cli.py local deploy
```

To run tests
```sh
$ python3 bin/spotlight-cli.py local test
```
