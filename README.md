# Setup

```sh
$ git clone git@github.com:juannyG/coms6998.git
$ cd coms6998
$ yarn install
$ cd packages/foundry
$ cp .env.example .env

# Open a new tab
$ yarn chain

# Open a new tab
$ yarn deploy

# Open a new tab
$ yarn start
```

To interact with spotlight, go to http://localhost:3000/spotlight
